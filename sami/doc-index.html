SqlSelectResult::getRows() — Method in class SqlSelectResult
Returns an array of rows. Each row is an array with keys as the (virtual) column name and values as the cell value. The type of cell values are juggled with the following special rules: - TINYINT(1) and BIT(1) in MySQL are expressed in bool - Signed long long, a.k.a. BIGINT [SIGNED], i.e. 64-bit unsigned integers, are expressed in int, because PocketMine only supports 64-bit machines.
